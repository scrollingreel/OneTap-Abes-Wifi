# OneTap 🔐📶

OneTap is a smart and lightweight Wi-Fi auto-login solution built specifically for ABES campus networks. Designed using Python and Flask, this project automates the repetitive login process with just **one tap**, ensuring fast, seamless, and secure connectivity.

## 🔧 Features
- 🔐 Auto-authentication to ABES Wi-Fi portal
- ⚡️ One-tap login interface
- 🌐 Built with Flask (Python)
- 💡 Lightweight and fast execution
- 🛡️ Secure credential handling

## 📚 Tech Stack
- **Python**
- **Flask**
- **Requests** (for handling HTTP logins)
- **HTML/CSS** (for frontend UI, if any)

## 🎯 Use Case
Ideal for students tired of logging into the ABES Wi-Fi portal every time they connect. Just launch OneTap and you're instantly online.
